COM 포트 프린터 설정
⚠️ 주의사항:
• Chrome 89+ 버전 필요
• HTTPS 또는 localhost에서만 동작
• 사용자가 직접 포트를 선택해야 함
⚠️ 한글 폰트 문제 해결:
• 대부분의 영수증 프린터는 한글 지원이 제한적입니다
• 폰트가 깨질 경우 영문 모드를 사용하세요
• 프린터 펌웨어에서 한글 폰트 지원 여부를 확인하세요
프린터별 한글 지원 명령어가 다를 수 있습니다.
매뉴얼을 참고하여 설정을 조정하세요.
1. Electron 앱으로 변환
npm install electron npm install serialport // Node.js SerialPort 모듈 사용 가능
2. Node.js 로컬 서버
// server.js const express = require('express'); const SerialPort = require('serialport'); const app = express(); app.post('/print', (req, res) => { const port = new SerialPort('COM1', { baudRate: 9600 }); port.write(req.body.data); });
3. Chrome 확장 프로그램
// manifest.json { "permissions": ["serial"], "background": { "scripts": ["background.js"] } }
