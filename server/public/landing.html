번호표 수기 작성

종이 번호표를 직접 뽑아 나눠주고, 언제 불러야 할지 직접 챙겨야 합니다.

✅

태블릿 터치 한 번으로 발급

고객이 직접 터치하거나 QR로 접수. 사장님은 호출 버튼만 누르면 됩니다.

📢

"OO번 손님!" 직접 호출

주방 일 하면서 손님을 큰 소리로 부르거나 놓치는 경우가 생깁니다.

✅

알림톡으로 자동 호출

전화번호 입력 시 순번이 되면 알림톡으로 자동 안내. 놓칠 걱정 없습니다.

주요 기능

필요한 것만, 딱 3가지

🎫

번호 발급

태블릿 터치 또는 QR 셀프 접수로 번호 발급. 음식점형 메뉴 선택도 지원합니다.

📣

호출 & 알림톡

버튼 하나로 다음 순번 호출. 알림톡 실패 시 자동 SMS로 전환됩니다.

📊

운영 현황

오늘 발급 수, 대기 현황, 발송 로그를 관리자 화면에서 한눈에 확인합니다.

가격

쓴 만큼만 내세요

기본 구독료 + 실제 발송된 메시지 건당 과금. 쓰지 않으면 추가 비용 없습니다.

월정액

기본 구독료 + 메시지 건당 과금 (알림톡 / SMS)

  • 샌드박스 트라이얼 — 실제 메시지 없이 전체 흐름 체험
  • 알림톡 우선 발송 · 실패 시 자동 SMS 전환
  • 성공 발송 건만 과금 (실패 건 제외)
  • 다창구·디스플레이 확장 가능 (Standard)

정확한 요금은 가입 후 관리자 화면에서 확인하실 수 있습니다.
